## STEP 5 — Google Cloud Storage (GCS)

### 5a. Creaza proiectul nou in GCP
  1. Mergi la: https://console.cloud.google.com
  2. Click pe "Select a project" -> "NEW PROJECT"
  3. Nume: "disertatie-federated-learning"
  4. Click CREATE

### 5b. Instaleaza Google Cloud CLI (o singura data)
  Windows: https://cloud.google.com/sdk/docs/install
  
  Dupa instalare, in terminal:
    gcloud init
    gcloud auth application-default login
    # Se deschide browser -> te autentifici cu contul universitatii

### 5c. Configureaza setup_gcs.py
  Deschide setup_gcs.py si modifica:
    GCS_PROJECT_ID  = "disertatie-federated-learning"  # ID-ul proiectului tau
    GCS_BUCKET_NAME = "edu-federated-data-mihaiela"    # Nume unic global!

### 5d. Incarca datele
  python setup_gcs.py --create-bucket   # Creeaza bucket-ul
  python setup_gcs.py --upload          # Incarca data/raw/ pe GCS
  python setup_gcs.py --verify          # Verifica ce e acolo

### 5e. Verifica in browser
  https://console.cloud.google.com/storage/browser/edu-federated-data-mihaiela
